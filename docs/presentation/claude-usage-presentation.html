ما هو Claude Code؟

"وكيل برمجي يعمل داخل الطرفية (Terminal) ويتفاعل مباشرة مع ملفاتك ومستودعك."

🖥️ أين يعمل؟

CLI، تطبيق سطح المكتب، الويب (claude.ai/code)، وإضافات IDE

🔧 ما الذي يفعله؟

يقرأ ويعدّل الكود، ينفّذ أوامر shell، يبحث في المشروع، ويُنشئ Commits و PRs

🔐 آمن بالتصميم

كل أداة تحتاج إذنًا — لا يُنفّذ أمرًا خطيرًا دون موافقتك

🧩 قابل للتوسعة

Skills، MCP Servers، Hooks، Subagents — كل شيء يُضاف ولا يُكتب من الصفر

الأدوات الأساسية

الأداةالاستخدام
Read / Write / Editقراءة وتعديل الملفات بدقة
Glob / Grepالبحث عن ملفات أو نصوص في المشروع
Bashتنفيذ أوامر الطرفية (git، npm، pytest...)
Agentتفويض مهام فرعية معقدة لوكيل متخصص
WebFetch / WebSearchجلب معلومات حديثة من الإنترنت

نظام Skills

وحدات معرفة جاهزة تُفعَّل تلقائيًا حسب نية الطلب — أو يدويًا عبر /اسم-المهارة.

1
كل Skill = مجلد فيه SKILL.md يحتوي وصفًا ومحفزات تشغيل
2
تتنافس الأوصاف على نية المستخدم — التكرار يسبب "اختيار المهارة الخاطئة"
3
أدوات مثل skill-doctor تفحص مكتبة المهارات وتكتشف التعارضات
4
يمكنك بناء مهاراتك الخاصة لمهام متكررة في مشروعك

توسيع القدرات: MCP و Subagents

🔌 MCP Servers

بروتوكول موحّد يصل Claude بأدوات وخدمات خارجية: GitHub، Slack، Notion، قواعد البيانات...

🤖 Subagents

وكلاء متخصصون (بحث، مراجعة كود، اختبار) يعملون بسياق مستقل لحماية النافذة الأساسية

🪝 Hooks

سلوكيات تلقائية مرتبطة بأحداث: قبل/بعد تنفيذ أداة، عند بدء الجلسة...

⚙️ Settings

تخصيص الصلاحيات، النماذج، والأدوات المسموح بها عبر settings.json

أفضل ممارسات كتابة الطلبات

  • ✅ كن محددًا: اذكر الملف، السطر، أو السلوك المتوقع
  • ✅ قسّم المهام الكبيرة إلى خطوات واضحة
  • ✅ استخدم CLAUDE.md لتوثيق قواعد المشروع الثابتة
  • ✅ راجع الفروقات (diff) قبل الموافقة على التغييرات الحساسة
  • 🚫 لا تطلب تنفيذ عمليات هدامة دون تأكيد صريح

مثال عملي

"أضف فحص صحة (health check) لتطبيق Node.js"

1
يفحص Claude بنية المشروع عبر Glob و Grep
2
يقترح خطة: مسار /health + ملف اختبار
3
ينفّذ التعديلات بـ Edit ويشغّل الاختبارات بـ Bash
4
يلخّص التغييرات ويترك القرار النهائي للـ commit لك

الخلاصة

Claude Code ليس مجرد أداة توليد كود — بل وكيل يعمل معك في بيئتك الحقيقية، قابل للتوسعة بلا حدود عبر Skills وMCP، ومصمم ليكون شفافًا وآمنًا في كل خطوة.
